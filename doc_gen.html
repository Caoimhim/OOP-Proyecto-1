Programa: HTML Documentator

Descripición: Crea Documentación de archivos en HTML
Autor: Kevin Chinchilla
Fecha de elaboración: 15 de febrero, 2019

Función: parseLine

Descripición: Revisa si la línea tiene una tag de documentación, y elimina los espacios en blanco antes de la tag si existen
Parametros: La string a revisar
Valor de retorno: La línea, si contiene el tag, o nulo, si no

Función: findComments

Descripición: Busca comentarios en el formato correcto para documentar
Parametros: Un arreglo de strings donde se guardaran los comentarios para procesar, y el tamaño de este arreglo
Valor de retorno: Verdadero, si todo sale bien, falso si no encuentra el archivo a revisar

Función: findTitle

Descripición: Busca la tag que contiene el nombre del programa
Parametros: El arreglo de strings con los comentarios a procesar, y el tamaño de este.
Valor de retorno: La posición del arreglo donde está el nombre del programa, o -1 si no se euncuentra un nombre para el programa

Función: writeSection

Descripición: Escribe una sección de docmentación que incluyen autor, fecha de creación, parámetros y valor de retorno
Parametros: El stream para escribir el archivo, la posición del arreglo dónde inicia la sección, y el arreglo con los comentarios a procesar
Valor de retorno: NONE

Función: writeHTML

Descripición: Escribe rl archivo HTML con la documentación
Parametros: El arreglo con los comentarios a procesar, y el tamaño de este
Valor de retorno: NONE

Función: main

Descripición: Llama todas las otras funciones
Parametros: Se puede pasar como parametro el nombre del programa a revisar
Valor de retorno: 0
